# 3fc
Web application for managing 3FC games and web app

## Developing

* Use make and other structured commands (implement if needed)

## Backlog

The implementation backlog for M0-M4 lives in:

- `docs/backlog/backlog.json` (canonical source)
- `docs/backlog/backlog.md` (generated summary)
- `docs/backlog/README.md` (workflow)

Common commands:

```bash
make backlog-validate
make backlog-export
make backlog-sync-dry REPO=<owner/repo>
make backlog-sync REPO=<owner/repo>
```
